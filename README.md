# Pollution Simulation

Численное моделирование распространения загрязнения воздуха в городской среде с учётом ветра и зданий-препятствий.

## Описание задачи

Решается уравнение конвекции-диффузии на двумерной равномерной сетке размером **300×300 м**. На сетке расположены 24 здания (корпуса 100–130), вокруг которых обтекает поток загрязнённого воздуха. Загрязнение поступает с левой границы и переносится ветром, одновременно рассеиваясь за счёт диффузии.

## Численные методы

### Диффузия — ADI (Alternating Direction Implicit)

Метод переменных направлений: на каждом шаге по времени задача расщепляется на два полушага — сначала неявно по X, затем по Y. Это позволяет решать трёхдиагональные системы вместо полной двумерной системы.

- Системы решаются методом прогонки Томаса (пакетная версия для всех строк/столбцов одновременно)
- Сегменты между зданиями предвычисляются заранее для ускорения

### Конвекция — Upwind-схема

Явная схема против потока: значение берётся с той стороны, откуда дует ветер. Направление определяется знаком `lambda1` (по X) и `lambda2` (по Y).

- Учитывается диагональный сосед для корректной обработки углов зданий
- Автоматически проверяется условие устойчивости CFL: `|λ₁|·τ/h + |λ₂|·τ/h ≤ 1`

### Шаг по времени

Выбирается автоматически из условий устойчивости:
- `τ = h / max(|λ₁|, |λ₂|)` — ограничение по конвекции
- При малых скоростях: `τ = h / 2π`

## Граничные условия

| Граница | Условие |
|---------|---------|
| Левая (x=0) | Дирихле: `u = g1` (источник загрязнения) |
| Правая (x=L) | Неймана: `u[-1] = u[-2]` (свободный выток) |
| Нижняя и верхняя | Неймана: нулевой нормальный поток |
| Здания | `u = 0` (непроницаемые препятствия) |

## Параметры запуска

Создай файл `input.txt` с одной строкой:

```
M eps g1 lambda1 lambda2 kappa T
```

| Параметр | Тип | Описание |
|----------|-----|----------|
| `M` | int | Число узлов сетки по каждому направлению |
| `eps` | float | Точность (зарезервировано) |
| `g1` | float | Концентрация загрязнения на входе (левая граница) |
| `lambda1` | float | Скорость ветра по оси X, м/с |
| `lambda2` | float | Скорость ветра по оси Y, м/с |
| `kappa` | float | Коэффициент диффузии, м²/с |
| `T` | float | Время моделирования, с |

**Пример `input.txt`:**
```
101 1e-6 1.0 5.0 2.0 0.5 10.0
```

## Запуск

```bash
pip install numpy matplotlib
python Pollution_simulation.py
```


### `picture.png`

Тепловая карта концентрации загрязнения: здания закрашены серым, воздушные узлы — по шкале `viridis` от нуля до максимума.

## Структура кода

| Функция | Описание |
|---------|----------|
| `build_solid_mask` | Создаёт булеву маску зданий на сетке |
| `precompute_segments` | Предвычисляет непрерывные воздушные сегменты для ADI |
| `make_tridiagonal` | Строит трёхдиагональную матрицу для прогонки |
| `thomas_batch` | Пакетная прогонка Томаса для всех сегментов |
| `diffusion_adi` | Один шаг диффузии методом ADI |
| `convection_upwind_formula` | Один шаг конвекции upwind-схемой |
| `solve` | Основной цикл по времени |
| `format_output` | Формирует строку для `output.txt` |
| `save_picture` / `main` | Сохраняет `picture.png` |

## Технологии
